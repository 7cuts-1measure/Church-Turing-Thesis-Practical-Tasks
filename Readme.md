# Кластер

## Описание

Кластер состоит из **5 узлов**, выстроенных в шину *кольцо*.
Раз в некоторое время необходимо передать данные с узла *i* на узел *i+1*.

Перекачкой данных занимается **worker** (между каждыми соседними узлами стоит свой worker).
Он подключается к двум соседним узлам и после этого передаёт данные.

По техническим причинам в один момент времени к одному узлу может быть присоединён **только один worker**.



## Задача

Вам предлагается программа на **C** [cluster.c](./cluster.c), которая описывает данный процесс.

Каждому *worker* соответствует свой поток исполнения.

Необходимо:

* Определить работоспособность протокола взаимодействия для *worker*’ов при помощи модели **TLA+**
* Если протокол содержит ошибки:

  * Привести пример **некорректной трассы исполнения**
  * Предложить **исправленный вариант протокола**
  * Проверить исправленный вариант при помощи модели **TLA+**



## Формат решения

В качестве решения необходимо предоставить:

* файл со **спецификацией алгоритма**
* **конфигурацию запуска TLC**
* отдельный файл с примером **трассы исполнения модели**

Если алгоритм некорректен:

* текстовое **описание проблемы в алгоритме**
* файл со **спецификацией нового алгоритма**
* **конфигурацию запуска TLC**
* отдельный файл с примером **трассы исполнения модели нового алгоритма**
